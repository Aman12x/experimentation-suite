## ðŸš€ Quick Start (5 minutes)

### Step 1: Extract Files
Extract this ZIP to a folder of your choice

### Step 2: Install Dependencies
```bash
# Open terminal in the extracted folder
pip install -r requirements.txt
```

### Step 3: Run the App
```bash
streamlit run app.py
```

Your browser will open to http://localhost:8501

### Step 4: Try It Out
1. Click "Browse files" in the sidebar
2. Upload `sample_ab_test_data.csv`
3. Go to "A/B Testing" tab
4. Select:
   - Group: `group`
   - Metric: `revenue`
   - Test: `T-Test`
5. Click "Run A/B Test"

## ðŸ“Š Sample Datasets Explained

## ðŸ†˜ Troubleshooting

### "streamlit: command not found"
```bash
pip install --upgrade streamlit
```

### "ModuleNotFoundError: No module named 'scipy'"
```bash
pip install --upgrade -r requirements.txt
```

### Port already in use
```bash
streamlit run app.py --server.port 8502
```

### Can't see modules folder
The ZIP contains folders! Make sure you:
1. Extract the ENTIRE ZIP (not just some files)
2. Extract to a folder (not just "open" the ZIP)
3. Check that `modules/` and `utils/` folders exist
